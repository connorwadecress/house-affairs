# House Affairs

House Affairs is a Pretoria-based house music collective and registered business in its earliest operating phase.

This repository is the operating memory for the collective: vision, strategy, research, content planning, venue outreach, correspondence, agentic operations, business notes, and future technical work.

Start here:

- `wiki/00 - Meta/Open Questions.md` - unknowns that should not be guessed.
- `wiki/01 - Vision & Strategy/Mission.md` - canonical mission draft.
- `wiki/01 - Vision & Strategy/Roadmap.md` - baseline roadmap.
